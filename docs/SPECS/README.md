# Specs Estratégicos

Esta pasta concentra specs de crescimento, monetização, IA, conteúdo, SEO e estratégia de longo prazo do PetCareTutor.com.

## Índice

- `GROWTH.md` — funil de aquisição, canais e métricas.
- `MONETIZATION.md` — modelo de afiliados e regras editoriais.
- `AI_CONTENT_INTELLIGENCE.md` — motor futuro de inteligência de conteúdo com IA.
- `CONTENT.md` — linhas editoriais, tom de voz e relação com afiliados.
- `SEO.md` — clusters, páginas programáticas e métricas orgânicas.
- `LONG_TERM_STRATEGY.md` — tese de longo prazo, riscos e indicadores para captação.

## Regra central

Nenhum spec desta pasta deve orientar marketplace, venda direta de animais, intermediação de adoções ou comunicação direta entre usuários.
